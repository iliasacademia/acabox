# Results Section Conventions

## Principles

1. **Describe, don't interpret (with one exception).** Results presents what was found. Save broad interpretation and connection to other work for Discussion. Statements like "This is consistent with the hypothesis that..." belong in Discussion. However, narrow data-proximate inference is appropriate in Results: "These data indicate that protein X is required for migration" is a direct conclusion from the data shown, not a broader claim.

2. **Anchor text to figures and tables.** Each paragraph should reference a specific figure or table. Use the format "(Figure X)" with consecutive numbering in order of first appearance. Never reference figures by filename.

3. **Follow the biology paragraph structure.** Each Results paragraph should follow this sequence:
   - **Rationale:** State the specific goal of the sub-experiment (the "why")
   - **Approach:** Briefly mention the technique or assay used (the "how")
   - **Finding:** State the primary observation or trend (the "what")
   - **Reference:** Cite the specific figure or table (the "evidence")
   - **Inference:** Provide a narrow, immediate conclusion from the data shown (the "so what")

4. **Include specific quantitative results.** Report actual values, effect sizes, confidence intervals, and statistical tests. Vague language ("there was a significant difference") without numbers is insufficient.

5. **Match the Methods structure.** Results should follow the same logical sequence as Methods so readers can trace each analysis to its procedure.

6. **No citations in Results** unless referencing a method, dataset, or tool by name. Results reports your own data, not others' work.
